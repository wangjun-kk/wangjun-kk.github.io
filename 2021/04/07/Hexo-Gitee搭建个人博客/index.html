Hexo+Gitee搭建个人博客

环境要求

  • Git

  • NodeJs

  • gitee账户

步骤

  1. 全局安装hexo

    npm install -g hexo-cli

  2. 准备blog项目的目录下初始化项目【wangjun-blog】

    hexo init wangjun-blog

  3. 进入项目根目录

    cd wangjun-blog

    npm install

  4. 本地开启调试服务

    hexo s

hexo命令

  • 写文章 - hexo new "文章名字"

  • 清理缓存 - hexo clean

  • 生成页面 - hexo g

  • git上传 - hexo d

    此处上传至gitee (也可选择github等，具体在配置文件_config.yml中配置)

部署到gitee

  1. 安装git插件 - npm install hexo-deployer-git

  2. 配置文件中配置

  3. hexo g

  4. hexo d
